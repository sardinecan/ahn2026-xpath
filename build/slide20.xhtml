Shorthand

XPath possède une syntaxe abréviée (shorthand) pour certaines expressions :

  • l’axe child:: est facultatif ;
  • l’axe attribute:: peut être remplacé par le symbole @ ;
  • l’axe parent:: peut être noté .. ;
  • l’axe descendant-or-self:: est abrévié en // (autrement dit, n’importe où dans le document).
  • Le nœud courant est abrévié .

                        
/child::body/child::div = /body/div
/child::body/child::div/attribute::type = /body/div/@type
/child::body/descendant::p = /body//p
