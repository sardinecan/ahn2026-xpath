Les nœuds

Bien évidemment, on peut chercher autre chose que des nœuds éléments, par exemple des attributs, ou du texte.

Il existe 7 types de nœuds :

  • le nœud racine (≠ de l’élément racine, on parle aussi de nœud document), il est unique (/)
  • les nœuds éléments
  • les nœuds attributs (axe attribute::)
  • les nœuds texte (text())
  • les nœuds commentaires (comment())
  • les nœuds instructions de traitement (processing-instruction())
  • les nœuds d’espace de nom

Les Jokers

  • * : tout nœud élément
  • node() : n’importe quel nœud (sauf les attributs)
  • @* : tout nœud attribut

Les opérateurs de séquence

  • | ou union : union
  • intersect : intersection
  • except : exclusion
  • () : combinaison
